Detailed Description

A representation of a box that is aligned with the x, y, and z axes.

Such a box is useful for making rough intersection calculations for an initial check, before performing more in-depth intersection checks. For example, a large model may contain one of these around the entire model, and – if an intersection occurs with this box – a more complicated intersection may be performed on all sub-parts of the model.
