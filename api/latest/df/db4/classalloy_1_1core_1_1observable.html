Detailed Description

An object in the system that may be observed by an observer_ptr.

Observerables contain extra data to be aware of their own observables, which allows the observing observer_ptr's to be nulled-out after destruction.

Example usage:

class some_lifetime_object : public observable
{
public:
some_lifetime_object() = default;
...
};
