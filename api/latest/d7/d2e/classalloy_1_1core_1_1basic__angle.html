Detailed Description

template<typename AngleUnit>
class alloy::core::basic_angle< AngleUnit >

Represents an angle as a generic base type.

The basic_angle class is provided as a utility for creating other angles, through the use of the AngleUnit template parameter.

Template Parameters
AngleUnitthe angle unit which specifies the value of one full revolution as a real
